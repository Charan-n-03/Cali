# 🏋️ CALI — Calisthenics Training Website

**CALI** is a modern, responsive, and visually rich single-page website dedicated to **Calisthenics training**.  
It educates users about calisthenics, showcases exercises by muscle groups, explains training methods, and allows users to download a **Beginner-to-Advanced workout plan**.

The website is built using **HTML, Tailwind CSS, and JavaScript**, focusing on clean design, smooth animations, and an immersive user experience.

---

## 🌐 Live Preview
> Open `Cali.html` directly in your browser  
(No backend or server setup required)

---

## ✨ Features

- 🎨 Premium dark-themed UI with custom fonts and animations  
- 📱 Fully responsive (desktop, tablet, mobile)
- 🧭 Smooth navigation with scroll-based sections
- 📘 Beginner → Advanced Calisthenics guidance
- 🏋️ Exercise tabs by muscle group:
  - Chest
  - Back
  - Shoulders
  - Triceps
  - Biceps
  - Core
  - Legs
- 🧠 Training methodologies explained:
  - Hypertrophy
  - Maximum Strength
  - Skill Training
  - Muscular Endurance
- 📄 Downloadable workout plan (modal popup)
- 🖨️ Printable workout plan layout
- ⚡ Smooth reveal animations & hover effects
- ♿ Accessibility-friendly structure (ARIA labels, semantic sections)

## 🚀 How to Run the Project

1. Download or clone this repository
2. Locate the file `Cali.html`
3. Double-click or open it in any modern browser (Chrome, Edge, Firefox)
4. That’s it — no installation required 🎉

---

## 🖨️ Printing the Workout Plan

- Click **“Download Workout Plan”**
- A modal opens with a structured workout table
- Use **Ctrl + P / Cmd + P** to print
- Print-friendly styles are already included
